Tools

I develop(ed) a number of research prototypes. Please contact me if you have questions about one them. More tools are to come soon!

The PCW prototype aims at experimenting with an original adaptation of counter abstraction for the verification of parameterized systems (see the LATA'14 "ordered counter abstraction" paper).

The UNDIP prototype enables the verification of safety properties for Unbounded Distributed Parametreized systems. UNDIP is a generalization of PFS which verifies the same type of properties for Parameterized Finite state Systems.
